The length-property: length \9; hack

Last updated 2016-01-23.

The hack

.your-selector-here {
  some-length-property-here: somelength \9;
}

…where somelength is a CSS length value and some-length-property-here is a CSS property whose value can be a CSS length.

Source: Browser CSS Hacks by Paul Irish

Important note! This article only applies to the specific properties mentioned below and only applies when there is no space between somelength and  \9. Adding spaces or using different properties may change the compatibility of the hack.
You have been warned.
